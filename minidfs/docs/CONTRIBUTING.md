# Contributing to MiniDFS

Thanks for your interest in contributing! Please follow these basics:

- Discuss large changes in an issue first.
- Keep PRs focused and small; include tests where applicable.
- Match existing code style and C++20 usage.
- Ensure the project builds with CMake and vcpkg on Windows.
- Run `minidfs_tests` locally and include results in the PR description.

For questions or clarifications, contact the maintainer at mattdev727@gmail.com.
